# ALCOMD3 v3.0.0

## English

ALCOMD3 3.0.0 is the first public release from the independent ALCOMD3 repository.

### Application updates

- Introduces the first public ALCOMD3 application release.

### Installation and upgrade

- Provides release packages for Windows x64, macOS Apple Silicon, and Linux x86_64 together with stable automatic update metadata.

### Compatibility and security

- Uses ALCOMD3's own update endpoint, embedded public key, and signed update payloads.
