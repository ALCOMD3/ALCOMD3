# ALCOMD3 v2.2.0-beta.6

## English

This beta adds extension management and drag-and-drop sidebar ordering, while improving the reliability of contributor data shown by ALCOMD3.

### Application updates

- Added an Extensions page for managing installable application extensions and controlling whether installed extensions appear in the sidebar.
- Added drag-and-drop ordering for installed and visible sidebar items, with matching icons and persistent saved order.
- Kept built-in navigation items separate from installable extension management, while the Extensions entry and version information remain fixed at the bottom of the sidebar.

### Website updates

- Fixed the deployed contributor data endpoint and synchronized contributor information with the ALCOMD3 GitHub repository contributor page.

### Installation and upgrade

- This version has no user-visible installation or upgrade changes.

### Compatibility and security

- This version has no user-visible compatibility, security, or known-issue changes.

## 日本語

このベータ版では、拡張機能の管理とサイドバーのドラッグ＆ドロップ並べ替えを追加し、ALCOMD3 が表示するコントリビューターデータの信頼性を向上しました。

### アプリの更新

- インストール可能なアプリ拡張機能を管理し、インストール済み拡張機能をサイドバーに表示するかどうかを設定できる拡張機能ページを追加しました。
- インストール済みで表示中のサイドバー項目を、対応するアイコン付きでドラッグ＆ドロップ並べ替えできるようにし、保存した順序が維持されるようにしました。
- 組み込みナビゲーション項目をインストール可能な拡張機能の管理対象から分離し、拡張機能入口とバージョン情報はサイドバー下部に固定しました。

### Web サイトの更新

- 公開環境のコントリビューターデータエンドポイントを修正し、ALCOMD3 GitHub リポジトリのコントリビューターページと情報を同期するようにしました。

### インストールとアップグレード

- このバージョンには、ユーザーに見えるインストールまたはアップグレードの変更はありません。

### 互換性とセキュリティ

- このバージョンには、ユーザーに見える互換性、セキュリティ、既知の問題の変更はありません。
