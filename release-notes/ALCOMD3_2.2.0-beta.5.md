# ALCOMD3 v2.2.0-beta.5

## English

This beta adds live ALCOMD3 contributor credits to the application and website, while refreshing the public project and MCP documentation with clearer localized entry points.

### Application updates

- Settings now includes a contributor avatar wall loaded from the ALCOMD3 GitHub repository, with contributor names available on hover and each avatar linked to its GitHub profile.

### Website updates

- The homepage now shows the current contributors from the ALCOMD3 GitHub repository as linked avatars. The section remains hidden when live data cannot be loaded, so the website does not display a stale fallback list.
- The public README and MCP guidance have been refreshed, and localized documentation has been reorganized under clearer language-specific paths.

### Installation and upgrade

- This version has no user-visible installation or upgrade changes.

### Compatibility and security

- This version has no user-visible compatibility, security, or known-issue changes.

## 日本語

このベータ版では、アプリと Web サイトに ALCOMD3 のコントリビューター情報をリアルタイムで表示する機能を追加し、公開プロジェクト文書と MCP ガイドを更新して、多言語の入口を分かりやすく整理しました。

### アプリの更新

- 設定画面に、ALCOMD3 の GitHub リポジトリから読み込むコントリビューターのアバター一覧を追加しました。ホバーで名前を確認でき、各アバターから GitHub プロフィールを開けます。

### Web サイトの更新

- ホームページに、ALCOMD3 の GitHub リポジトリの現在のコントリビューターをリンク付きアバターで表示するようにしました。リアルタイムデータを取得できない場合はセクションを非表示にし、古いフォールバック一覧を表示しません。
- 公開 README と MCP ガイドを更新し、多言語ドキュメントを言語ごとの分かりやすいパスに再編成しました。

### インストールとアップグレード

- このバージョンには、ユーザーに見えるインストールまたはアップグレードの変更はありません。

### 互換性とセキュリティ

- このバージョンには、ユーザーに見える互換性、セキュリティ、既知の問題の変更はありません。

## 中文

### 应用更新

- 设置页新增从 ALCOMD3 GitHub 仓库加载的贡献者头像墙；悬停头像可查看名称，点击头像可打开对应的 GitHub 主页。

- 官网首页新增 ALCOMD3 GitHub 仓库当前贡献者头像墙。实时数据加载失败时会隐藏整个板块，不展示可能过时的回退名单。
- 更新公开 README 与 MCP 指南，并将多语言文档整理到更清晰的语言专属路径下。
