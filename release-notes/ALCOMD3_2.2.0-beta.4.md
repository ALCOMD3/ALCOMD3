# ALCOMD3 v2.2.0-beta.4

## English

This beta restores the intended automatic-update flow: updates download without confirmation when automatic updates are enabled, remain ready for installation, and install on the next launch if deferred, while manually confirmed updates continue directly into installation.

### Application updates

- Automatic updates now start the shared update download correctly, and download progress remains available from the update indicator and progress dialog.
- The automatic-update setting now accurately explains that enabled updates download automatically and downloaded updates install on the next launch.

### Website updates

- This version has no user-visible website changes.

### Installation and upgrade

- Automatically downloaded updates no longer install immediately after the download completes. The completion dialog can install them on demand, or the app installs them automatically on the next launch.
- Manually confirmed updates continue from the shared download flow directly into installation.

### Compatibility and security

- This version has no user-visible compatibility, security, or known-issue changes.

## 日本語

このベータ版では、本来の自動更新フローを復元しました。自動更新が有効な場合は確認なしで更新をダウンロードし、インストール可能な状態で保持します。保留した更新は次回起動時にインストールされ、手動で確認した更新はダウンロード後にそのままインストールへ進みます。

### アプリの更新

- 自動更新が共通の更新ダウンロードを正しく開始するようになり、更新インジケーターと進行状況ダイアログからダウンロードの進行状況を確認できます。
- 自動更新の設定説明を修正し、有効時は更新を自動的にダウンロードし、ダウンロード済みの更新を次回起動時にインストールすることを明確にしました。

### Web サイトの更新

- このバージョンには、ユーザーに見える Web サイトの変更はありません。

### インストールとアップグレード

- 自動的にダウンロードした更新は、ダウンロード完了直後にインストールされなくなりました。完了ダイアログから任意のタイミングでインストールするか、次回起動時に自動的にインストールできます。
- 手動で確認した更新は、共通のダウンロードフローからそのままインストールへ進みます。

### 互換性とセキュリティ

- このバージョンには、ユーザーに見える互換性、セキュリティ、既知の問題の変更はありません。

## 中文

此测试版恢复了符合预期的自动更新流程：开启自动更新后，应用会免确认下载更新并保留为可安装状态；如果暂不安装，则在下次启动时安装。用户手动确认的更新仍会在下载完成后直接进入安装。

### 网站更新

- 此版本没有用户可见的网站变化。

### 安装与升级

- 自动下载的更新不再在下载完成后立即安装。用户可以在下载完成页按需安装，也可以留到下次启动时自动安装。
- 用户手动确认的更新仍会在共用下载流程完成后直接进入安装。

### 兼容性与安全

- 此版本没有用户可见的兼容性、安全或已知问题变化。
