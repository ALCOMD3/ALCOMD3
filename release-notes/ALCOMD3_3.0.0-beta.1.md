# ALCOMD3 v3.0.0-beta.1

## English

This beta starts ALCOMD3 3.0 from its independent organization repository and simplifies contributor data loading while preserving the current extension and sidebar experience.

### Application updates

- Contributor information is now requested directly from the ALCOMD3 GitHub repository contributors API and cached for 30 minutes, without generated snapshots or repository-maintained fallback data.

### Website updates

- Website repository, release, download, and contributor links now use the independent `ALCOMD3/ALCOMD3` repository.

### Installation and upgrade

- Release assets and updater metadata now use `ALCOMD3/ALCOMD3` as their only repository source, while the public updater endpoints remain under `alcomd3.cqmhv.com`.
- This version begins a new release line in the independent repository; no release assets are copied from the former repository.

### Compatibility and security

- This version has no user-visible compatibility, security, or known-issue changes.

## 日本語

このベータ版は、ALCOMD3 3.0 を独立した組織リポジトリから開始し、現在の拡張機能とサイドバーの操作性を維持しながら、コントリビューターデータの取得を簡素化します。

### アプリの更新

- コントリビューター情報は ALCOMD3 GitHub リポジトリの contributors API から直接取得し、30 分間キャッシュするようになりました。生成済みスナップショットやリポジトリ内のフォールバックデータは使用しません。

### Web サイトの更新

- Web サイトのリポジトリ、リリース、ダウンロード、コントリビューターへのリンクを、独立した `ALCOMD3/ALCOMD3` リポジトリに変更しました。

### インストールとアップグレード

- Release アセットと updater metadata は `ALCOMD3/ALCOMD3` のみをリポジトリソースとして使用し、公開 updater endpoint は引き続き `alcomd3.cqmhv.com` を使用します。
- このバージョンから独立リポジトリ上の新しいリリース系列を開始し、旧リポジトリの Release アセットはコピーしません。

### 互換性とセキュリティ

- このバージョンには、ユーザーに見える互換性、セキュリティ、既知の問題の変更はありません。
