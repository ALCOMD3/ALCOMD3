# Release assets

This directory stores reusable assets and notes for external release channels.
It is not part of the application build and should not contain runtime assets.

Current assets:

- `ALCOMD3-BOOTH-PSD.psd`: BOOTH page source image.
- `ALCOMD3-BOOTH-1.png`: BOOTH page exported image.
- `ALCOMD3-BOOTH-2.png`: BOOTH page exported image.

Keep channel-specific material here when it is useful for future releases.
Keep website runtime assets under `website/public/`, and app runtime icons under
`vrc-get-gui/icons/`.
